# automotive-ICE-modelling

A learning tool for Python programming and mechanical engineering concepts, with an emphasis on modelling and simulating internal combustion engines (ICE).

## Purpose

This project is designed as an educational resource for:
- Practicing and improving Python programming skills
- Gaining a deeper understanding of mechanical engineering, especially as it relates to internal combustion engine modeling

## Intended Audience

- Engineers seeking to reinforce or broaden their skills in simulation and coding
- Students learning about ICE and looking for practical tools to support coursework, projects, or self-study

## Features

- Written entirely in Python for accessibility and ease of customization
- Contains sample scripts and models relevant to ICE simulation and analysis
- Facilitates learning in both programming and mechanical system concepts

## Installation

### Prerequisites

- Python 3.x installed on your system  
- `pip` for managing Python packages

### Setup

1. **Clone the repository**
    ```sh
    git clone https://github.com/octarine123/automotive-ICE-modelling.git
    cd automotive-ICE-modelling
    ```

2. **Install requirements**
    ```sh
    pip install -r requirements.txt
    ```

## Usage

Browse the Python scripts and notebooks in the repository to explore different aspects of engine modeling. Each file contains comments and documentation to guide learning.
